40 /* scr_swap_files -- copy or move a file from one node to another
41  * if swap_type = COPY_FILES
42  * if file_send != NULL, send file_send to rank_send, who will make a copy,
43  * copy file from rank_recv if there is one to receive
44  * if swap_type = MOVE_FILES
45  * if file_send != NULL, move file_send to rank_send
46  * save file from rank_recv if there is one to receive
47  * To conserve space (e.g., RAM disc), if file_send exists,
48  * any incoming file will overwrite file_send in place, one block at a time.
49  * It is then truncated and renamed according the size and name of the incoming file,
50  * or it is deleted (moved) if there is no incoming file.
