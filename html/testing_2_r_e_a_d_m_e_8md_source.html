10 OVERVIEW
11 ---------------------------------
12 There are 2 ways to use the testing scripts in this directory:
13 fully automated testing and interactive testing.
14 
15 FULLY AUTOMATED TESTING
16 ---------------------------------
17 To use fully automated testing, simply run the
18  BUILD_AND_TEST
19 bash script with a single argument, the file to which you want output written.
20 The BUILD_AND_TEST script will build SCR and run the TEST python script as
21 a batch job, which will parse the output and determine whether or not the
22 test was passed. The last thing printed into the output file will be whether
23 the test passed or failed. If the output file is a relative path, the path
24 will be interpreted relative to the examples directory, found at
25 $SCR_INSTALL/share/scr/examples
26 
27 INTERACTIVE TESTING
28 ---------------------------------
29 To use interactive testing, run the
30  BUILD_FOR_TESTING
31 bash script with a single argument, the maximum length of the interative
32 testing session. The BUILD_FOR_TESTING script will build SCR and open
33 an mxterm allocation on 4 nodes for testing.
34 
35 Testing commands to copy into the interactive testing session can be found in the
36  TESTING.sh
37 and
38  TESTING.csh
39 scripts. These scripts are identical except for language. The TESTING.sh
40 script is for bash users, and the TESTING.csh script for csh users.
41 These commands are not required for interative testing, but if executed in
42 the presented order represent the same tests as the automated TEST script.
43 It is also possible to run the automated TEST python script from an interactive
44 testing session.

46 DISTRIBUTION
47 ---------------------------------
48 
49 To verify that current version of SCR is ready for distribution, run the DISTRIBUTION.sh script from the SCR root directory:
50 
51 ```
52  $ cd /path/to/scr/
53  $ ./testing/DISTRIBUTION.sh
54 ```
55 
56 Note that no environment variables need to be set.
57 A sucessful and valid distribution will end with the following message:
58 
59 ```
60 ===========================================
61 scr-1.1.8 archives ready for distribution:
62 scr-1.1.8.tar.gz
63 ===========================================
64 ```
